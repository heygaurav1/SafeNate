# SafeNet Web

A React web application built for Indian citizens to protect against cyber scams (Phishing links, fake UPI IDs, Scam SMS).

## Setup
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run start
```

### ML Engine
```bash
cd ml
pip install -r requirements.txt
uvicorn server:app --reload
```
